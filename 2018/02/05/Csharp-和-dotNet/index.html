.NET Framework类所在名称空间

名称空间 说 明
System-Collections 这是集合的根名称空间。子名称空间也包含集合，如System.Collections.Concurrent和 System.Collections. Generic
Systern.Data 这是访问数据库的名称空间。System.Data.SqlClient包含访问SQL Server的类
System.Diagnostics 这是诊断信息的根名称空间，如事件记录和跟踪(在System.Diagnostics.Tracing名称空间中）
System.Globalization 该名称空间包含的类用于全球化和本地化应用程序
System.10 这是文件10的名称空间，其中的类访问文件和目录，包括读取器、写入器和流
System.Net 这是核心网络的名称空间，比如访问DNS服务器，用System.Net.Sockets创建套接字
System.Threading 这是线程和任务的根名称空间。任务在System.Threading.Tasks中定义
System.Web 这是ASP.NET的根名称空间。在这个名称空间下面定义了许多子名称空间，如System.Web.UI、 System.Web.ULWebControls 和 System.Web.Hosting
System. Windows 这是用于带有WPF的Windows桌面应用程序的根名称空间。子名称空间有 System.Windows.Shapes、System.Windows.Data 和 System.Windows.Documents

.NET Core

.NET Core是新的开源.NET框架，运行库是CoreCLR库，包含集合类的框架、文件系统、控制台和XML都在CoreFX库中。.NET Core采用模块化设计，框架和应用程序一起交付，应用程序发布时与它需要的框架一起打包。它使用新的命令行程序.NET Core Command Line (CLR)。

程序集（Assembly）

.NET程序的库和可执行文件称为程序集（Assembly）。Assembly是预编译好的、完全自描述性的逻辑单元，它可以存储在多个文件中（动态程序集存储在内存中）。存储在多个文件中的程序集需要一个包含入口点的主文件来描述其他文件。可执行的程序集包含一个主程序入口点，库程序集不包含。

C#基本语法

  • 语句以分号;结尾，以花括号{}组合为块。
  • 单行注释//，多行注释/**/，///三斜杠用于生成XML说明文档。
  • 区分大小写。
  • 变量必须进行初始化。类和结构中的变量的默认值为0，方法中的局部变量必须进行显式初始化。
  • 可以通过var关键字进行类型推断。
  • 数据类型分为值类型和引用类型。

流程控制语句

  • if…else…
  • switch…case…
    • 前面的case语句被激活后，后面的不会被激活；
    • 一条case语句为空，可自动跳至下一条；
    • case语句的顺序不重要，不能有相同的case；
    • 可以使用字符串做测试变量；
  • for循环
  • while循环
  • do…while…循环
  • foreach循环
    • 用于实现了IEnumerable接口的集合对象；
    • 不能用于改变集合中的项；
  • goto
  • break
  • continue
  • return

XML说明文档

标 记 说 明
