人们常说“鱼和熊掌不可兼得”，学习一项技能所花费的时间和它给你带来的价值往往成正比。但是也有一些被人忽略的小技能，掌握它并不需要耗费太多的精力，但是熟练使用以后却能够让工作效率得到很大提高。这样高性价比的技能可千万不要错过。

#用双拼输入法输入汉字

文字输入是人们日常生活和办公中必不可少的一个环节，消耗了人们的大量时间。然而，怎样才能获得最高的文字录入效率呢？曾经，很多人推荐五笔。但是五笔输入法有一些明显的弊端。一是学习门槛高，需要记忆大量的编码，时间一长也容易遗忘。另一个是在非单纯录入文字而是边想边输入的情况下，五笔需要输入者分心去回忆汉字编码，不仅打断了创作思路，输入速度也会大幅降低。与此同时，得益于现在互联网技术的发展，拼音输入法利用海量词库和在线联想，输入准确率得到了极大提高。可以说词库和联想功能的强大与否直接决定了一个输入法的好坏，人们再也没有足够的动力抛弃拼音输入法了。那么我们的输入速度能不能在拼音输入法的基础上进一步提高呢？

答案当然是肯定的，而且这样的提高不需要付出太多的学习成本，基本可以做到一天上手，一周熟练掌握。提高的方法就是将大家普遍使用的全拼换成双拼。

通常一个汉字分为声母和韵母，全拼模式在输入时一般需要输入全部字母，或者输入词组的全部声母，而双拼输入法将两个以上字母组成的声母和韵母与单一键位对应，所有汉字无论原来拼音的长短，现在只需要输入两个字母即可，大大简化了按键次数，提升了效率。同时，输入字母的减少也让输错字母时的修改、定位变得更加方便。而且，由于每个汉字输入的字母数固定，不像全拼那样每个词组有不同的简化方式，双拼输入往往以词组为固定单位，每次输入差不多的字母即可按空格键将候选词上屏，有时能带来一种意想不到的节奏感。这种节奏感能进一步输入体验。

那么怎么使用双拼输入法呢？其实非常简单，不需要重新安装软件，因为绝大部分拼音输入法都自带双拼方案，只需要到设置菜单里选择就行了。现在的双拼方案有很多种，比如微软双拼、小鹤双拼、自然码等等。区别不大，选择一种熟练使用即可。笔者使用的是小鹤双拼方案。网上可以下载专门的双拼练习软件，基本一两个小时内就能熟悉大部分键位。一旦熟练以后，随着日常生活中的使用，会越来越熟练。

#手机上使用滑动输入法
双拼输入法虽然好，但是在手机上使用却面临一些问题。虽然搜狗之类的常用手机输入法也可以选择双拼方案，但是只能在全键盘模式下使用，受限于手机屏幕的尺寸，全键盘模式下容易误按，因此并不推荐在手机上使用双拼输入法。

手机上的输入有一个更好的替代方案，那就是使用滑动输入法。滑动输入法让你在输入的时候手指无需离开屏幕，直接在本来要按的按键之间滑动，输入法会自动判断你要输入的字母。滑动输入法除了能够降低误触的概率，减少疲劳感之外，每一次滑动实际上起到了自动分词的作用，提高了输入效率。

#用正则表达式处理文本
大家都用过Word等办公软件里的查找替换功能，可以方便的找出大量文字中的所需信息。但是这一功能非常简单，无法满足一些特殊需求。比如，如果你需要找出文本中的所有电话号码，怎么办呢？这个正则表达式就能派上用场了。

正则表达式的功能非常强大，但是语法也很令人头疼。如果你不是专业人士，只是用它来简化文字处理工作，可以尝试用一些现成的正则表达式处理软件。

#用Markdown快速写作
Markdown是一个简洁的文本标记语言，用于对文本快速进行格式化。

在这里你可以看到它的使用说明，非常简单，几分钟内就可以掌握它的基本语法。

我们为什么要学习使用Markdown呢？主要目的是让我们在电脑或网络上书写文字的时候能够专注于文字内容本身，而不是被各种格式甚至所羁绊。Markdown可以通过在文本内容中输入一些简单的字符生成格式化的文本，形成各级标题、有序无序列表、粗体、斜体、下划线等等。不需要将键盘上敲字敲得正起劲的手挪到鼠标上，点击各种图标来设置格式。现在有很多软件和网站都已经支持Markdown语法。比如为知笔记、简书，甚至可以通过在Chrome上安装Markdown Here插件，在网页文本框中直接将Markdown转化为html格式化语言。

笔者最近发现了一个小众而强大的Markdown编辑器“小书匠”，不仅支持增强的Markdown语言，还可以直接与各个笔记平台同步。界面美观，甚至提供了沉浸式的写作模式，令人爱不释手。希望大家去尝试一下。

#用远程桌面异地办公
我们经常会遇到这样的情况，在办公室处理了一大堆数据文件，然而天色已晚，想回宿舍继续搞，但是桌面上打开的这一堆程序怎么办？一个个保存到U盘太费时费力，只好放弃做一次学霸的机会，明天再战。又或者在图书馆，身边只有笔记本，但是自己的笔记本很渣，不如办公室那装满了各种软件的台式机给力，怎么才能让远在办公室的台式机发挥作用呢？这些情况下你就需要远程桌面软件来助你一臂之力了。

远程桌面的作用就是将一台电脑的屏幕画面通过网络传输到远处的另一台电脑上，在远处电脑上的所有键盘输入和鼠标移动等操作也可以传输到那台电脑。这样我们就可以远程利用一台电脑的全部机能和软件。

远程桌面客户端有很多，Radmin是Windows上比较好的选择。Anydesk则是一个比较轻量化的选择。

#用批处理来简化系统操作
如果你的工作经常需要用到一些软件，一一打开非常麻烦，又不想让它们同时开机启动，拖慢开机速度，可以写一个批处理脚本，双击之后就可以同时打开数个软件。
比如，笔者常用的这个脚本，可以同时打开QQ、微信、为知笔记等等，如果你想要打开其他软件，只需要把路径替换掉即可。

start “” “D:\SSVIP\ShadowsocksR-3.7.4.1最新版.exe” &
start “” “C:\Program Files (x86)\Tencent\QQLite\Bin\QQ.exe” &
start “” “D:\Program Files\Foxmail 7.2\Foxmail.exe”&
start “” “C:\Program Files (x86)\Tencent\WeChat\WeChat.exe”&
start “” “C:\Program Files\Listary\Listary.exe”&
start “” “C:\Program Files (x86)\WizNote\Wiz.exe”&
start “” “C:\Users\Jerry\AppData\Local\Wunderlist\Wunderlist.exe”&
